# Uzcat Enterprise
Uzcat is an advanced, bi-lingual (UZ/EN) network audit and analysis utility built for security professionals and penetration testers. It provides robust tools for network monitoring, port diagnostics, and traffic manipulation.

## Key Features
- **Multi-Client Architecture:** Asynchronous handling of concurrent connections.
- **Port Knocking:** Security mechanism to obscure ports from unauthorized access.
- **TTL Secure Timeout:** Automatic cleanup mechanism to prevent "zombie" connections.
- **Real-time HEX Analysis:** Visualize incoming/outgoing traffic in hex dump format.
- **Bi-lingual Interface:** Native support for English and Uzbek languages.
